# YunPu

云谱，一个自托管的中文个人CRM系统。

## 技术栈

- 前端：Vue.js + ElementPlus
- 后端：FastAPI + SQLAlchemy
- 数据库：SQLite（预留多数据库支持）
- 部署：Docker Compose

#### 登录页

- [ ] 用户名/密码登录

作为个人CRM系统，隐私保护是最重要的。
虽然不支持多用户，但是进入系统前需要先登录，防止被他人访问。
用户名和密码都是用户自定义的，系统不会存储任何个人信息。
默认用户名和密码应该在`.env`文件中配置。

#### 个人主页

- [ ] 显示用户信息
- [ ] 显示事件时间轴
- [ ] 显示日期提醒

事件时间轴显示个人的人生事迹，包括工作经历、教育经历、个人经历等。
日期提醒显示用户设置的日期提醒，例如生日、纪念日等。

#### 联系人列表

- [ ] 显示用户所有联系人

#### 联系人详情页

- [ ] 显示联系人基本信息
- [ ] 显示联系人事件时间轴
- [ ] 显示联系人日期提醒

#### 家谱页

- [ ] 显示用户所有谱系

通过D3.js/XYFlow/vue-flow等库，实现谱系的可视化展示。
