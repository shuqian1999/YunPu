# 云谱 - UI设计规范

## 设计理念

云谱的设计灵感来源于蓝天白云，传递清新、简洁、宁静的视觉体验。整体风格追求简约、自然、易用，让用户在管理人物、事件、提醒和家谱时感到舒适。

## 间距系统

基于 4px 基础单位，采用 4 的倍数进行间距设计。

| 间距名称 | 数值 | 用途 |
|----------|------|------|
| **极小** | 4px | 图标与文字间距 |
| **小** | 8px | 标签间距、紧凑布局 |
| **中** | 12px | 表单元素间距 |
| **标准** | 16px | 卡片内边距、段落间距 |
| **大** | 20px | 区块间距 |
| **超大** | 24px | 页面边距、主要区块间距 |
| **极大** | 32px | 页面顶部/底部边距 |

## 交互设计规范

### 悬停效果

| 元素 | 悬停效果 |
|------|----------|
| 按钮 | 背景色加深，轻微上浮 |
| 卡片 | 阴影加深，轻微上浮 |
| 链接 | 下划线，颜色加深 |
| 图标 | 颜色加深，轻微放大 |

### 点击效果

| 元素 | 点击效果 |
|------|----------|
| 按钮 | 背景色变深，轻微缩小 |
| 链接 | 颜色变深 |
| 图标 | 轻微缩小 |

### 加载状态

- 使用天蓝色加载动画
- 加载文字：`#909399`
- 遮罩层：`rgba(255, 255, 255, 0.8)`

### 空状态

- 空状态图标：浅灰色
- 空状态文字：`#909399`
- 引导按钮：主按钮样式

### 错误状态

- 错误提示：红色背景 + 白色文字
- 错误图标：红色
- 错误文字：`#F56C6C`

## 响应式设计

### 断点定义

| 断点名称 | 屏幕宽度 | 布局调整 |
|----------|----------|----------|
| **移动端** | < 768px | 单列布局，隐藏侧边栏 |
| **平板** | 768px - 1024px | 双列布局 |
| **桌面** | > 1024px | 三列布局，显示侧边栏 |

### 移动端适配

- 导航栏：简化为汉堡菜单
- 卡片布局：单列显示
- 字号：适当缩小
- 间距：适当缩小

## 无障碍设计

### 颜色对比度

- 主要文字与背景对比度 ≥ 4.5:1
- 大号文字与背景对比度 ≥ 3:1
- 链接与背景对比度 ≥ 3:1

### 键盘导航

- 所有交互元素支持 Tab 键导航
- 焦点状态使用天蓝色边框
- 快捷键提示清晰可见

### 屏幕阅读器

- 所有图片提供 alt 文本
- 表单元素提供 label
- 错误提示与表单元素关联
